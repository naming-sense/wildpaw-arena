# WILDPAW ARENA (가칭) - 웹 탑뷰 실시간 PvP 기획 패키지

## 0) 프로젝트 한 줄 요약
**귀엽지만 매혹적인 5등신 동물 히어로**가 총기와 특수능력으로 싸우는, **전술 중심 3:3 / 5:5 탑뷰 실시간 대난투 PvP 웹게임**.

- 레퍼런스 감성: 브롤스타즈의 직관성 + 팀 전술성과 개성 있는 히어로 플레이
- 기술 방향: **Three.js 기반 3D 클라이언트** + **C++(standalone Asio) 권위형(authoritative) 서버**
- 사업 목표: 가벼운 진입장벽 + 깊은 메타 + 코스메틱 중심 BM으로 대중성과 매출 동시 확보

## 1) 핵심 목표
1. **대중성**: 3분 내외 짧은 경기, 쉬운 조작, 시각적으로 명확한 정보 전달
2. **경쟁성**: 스킬-포지셔닝-조합 카운터가 살아있는 팀 전술
3. **확장성**: 히어로/맵/모드/이벤트를 시즌 단위로 빠르게 추가 가능
4. **수익성**: P2W 없이도 결제 전환이 가능한 코스메틱·배틀패스·이벤트 상점 구조

## 3) 우선 실행 순서(권장)
1. **전투 프로토타입(4~6주)**: 이동/사격/스킬/피격/사망 + 기본 네트워크 동기화
2. **세로 슬라이스(8~10주)**: 3개 맵 + 8히어로 + 3v3 랭크 루프
3. **라이브 준비(6~8주)**: 결제/배틀패스/로그/운영툴/AB 테스트
4. **소프트런칭(8주)**: 지표 기반 밸런싱/튜토리얼/매칭 품질 개선

## 4) 방향성 한마디
이 프로젝트의 승부 포인트는 **“짧고 시원한 손맛 + 팀 전술의 깊이 + 매력적인 캐릭터 수집/꾸미기”**입니다.

웹에서도 충분히 경쟁력 있는 실시간 PvP를 만들 수 있지만,
성공의 핵심은 **네트워크 품질(지연 보정) + 읽기 쉬운 전투 연출 + 시즌 운영력**입니다.
