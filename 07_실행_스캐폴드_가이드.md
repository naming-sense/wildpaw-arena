## 2) 현재 구현 범위
- Room 서버
  - WebSocket(Boost.Beast) 수신
  - `C2S_INPUT` decode → `RoomSimulation.pushInput` 연결
  - 30Hz authoritative tick
  - `S2C_SNAPSHOT_BASE`/`S2C_SNAPSHOT_DELTA` JSON 직렬화 후 전송
  - 거리 기반 interest filter 후 변경 엔티티 전송
- Client netcode
  - 입력 예측/복구 기본 함수
  - snapshot 보간 버퍼
  - WebSocket 클라이언트 래퍼
- 프로토콜
  - 메시지 타입/공통 헤더(공유 정의)
  - 임시 JSON envelope (추후 binary 전환 예정)

## 3) 실행 방법
```bash
cd server
cmake -S . -B build
cmake --build build -j
./build/room/wildpaw-room 7001
```
