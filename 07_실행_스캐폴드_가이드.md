## 2) 현재 구현 범위
- Room 서버: 30Hz 틱 루프 + 입력 버퍼 + 이동 시뮬 + delta 감지
- Client netcode: 입력 예측/복구 기본 함수 + snapshot 보간 버퍼
- 프로토콜: 메시지 타입/공통 헤더 뼈대
