# 03. 아트 · 월드 · UX 가이드

## 1) 아트 디렉션 키워드
- **키워드**: Cute / Chic / Tactical / Punchy
- **비율**: 5등신(머리 큼 + 팔다리 짧고 탄탄한 실루엣)
- **톤**: 밝은 채도 + 금속/네온 포인트 + 팀 컬러 대비 강함
- **목표 인상**: "귀엽고 보기 쉬운데, 전투는 프로처럼 날카롭다"

---

## 2) 캐릭터 시각 규칙

### 2-1. 실루엣 우선
- 0.3초 안에 역할군 구분 가능해야 함
- 무기 형태로 역할 식별 가능(샷건=짧고 두꺼움, DMR=길고 슬림)

### 2-2. "귀엽지만 매혹적인" 표현 가이드
- 표정/헤어/액세서리로 매력 표현
- 의상은 기능성 + 스타일 중심(과도한 노출 지양)
- 승리/대기 애니메이션에 개성 강조

### 2-3. 스킨 체계
- 일반(색변형) / 희귀(실루엣 변주) / 영웅(이펙트) / 전설(보이스·킬이펙트)

---

## 3) 맵 아트 방향

## 3-1. 시각 가독성 원칙
- 바닥은 중명도, 캐릭터와 투사체는 고채도 강조
- 위험 지대는 명확한 테두리 + 점멸 텔레그래프
- 엄폐물 높이/통과 가능 여부를 색/아이콘으로 통일

## 3-2. 초기 맵 테마 3종
1. **네온 정글 돔**: 식생+네온 장치, 부쉬 플레이 중심
2. **하버 메탈야드**: 컨테이너 엄폐 + 직선 교전 구간
3. **루프탑 카니발**: 점프 패드/낙하 리스크, 하이모빌리티 전장

---

## 4) UX/UI 설계

## 4-1. 전투 HUD 최소 구성
- 좌측: 체력/실드, 상태이상
- 우측: Q/E/R 쿨다운, 탄창/재장전
- 상단: 점수/남은 시간/목표 진행도
- 중앙: 조준점 + 피격 방향 인디케이터

## 4-2. 사용자 흐름
1. 로그인
2. 로비(히어로/스킨/상점/패스)
3. 큐 진입
4. 픽/밴(랭크에서 선택적 적용)
5. 매치
6. 결과/보상
7. 리매치

### UX 원칙
- 3클릭 이내 매치 진입
- 버튼 위치 고정(근육 기억 지원)
- 모바일 대응 시 버튼 영역 재배치(엄지 존)

---

## 5) 사운드/타격감 연출
- 히트 사운드 3단계(일반/치명/처치)
- 스킬 발동 시 주파수 대역 분리(중요 스킬이 소음에 묻히지 않게)
- 궁극기에는 고유 시그니처 사운드 + 화면 테두리 연출
- 카메라 쉐이크는 짧고 강하게(멀미 최소화)

---

## 6) 성능 친화 아트 파이프라인(웹 전용)
- 텍스처 아틀라스 적극 사용
- LOD 2~3단계
- 스켈레탈 애니메이션 본 수 제한(캐릭터당 60 이하 권장)
- 그림자 품질 옵션(OFF/LOW/MED/HIGH)
- 투사체/이펙트는 GPU 인스턴싱 우선

---

## 7) 접근성 디자인
- 색약 모드(3타입)
- UI 크기 조절(100/125/150%)
- 진동/쉐이크 강도 조절
- 텍스트 대비 WCAG 근접 기준 확보

---

## 8) QA 체크리스트 (아트/UX)
- 전투 중 적/아군 식별 오류율 < 3%
- 신규 유저 첫 매치 진입 시간 < 90초
- 평균 FPS 60(권장), 저사양 30 안정
- 스킬 경고 인지율(테스트) > 85%

---

## 9) 레퍼런스 컨셉 시안 반영 노트 (2026-02-14)

### 9-1. 시안에서 가져갈 강점
- 큰 눈/둥근 얼굴/짧은 체형으로 "귀여운 첫인상"이 매우 강함
- 전술 조끼, 무릎 보호대, 허벅지 홀스터 등으로 **전투 직군 캐릭터성**이 즉시 읽힘
- 화이트 퍼 + 카키 전술장비 대비가 좋아서 실루엣 가독성이 높음
- 헤드셋/마이크 등 커뮤니케이션 장비 디테일이 팀플레이 테마와 잘 맞음

### 9-2. 게임 적용 시 보정 포인트
- 탑뷰 시인성을 위해 상반신/무기 실루엣을 더 굵고 단순화
- 팀 컬러(아군/적군) 식별을 위해 조끼·암밴드·이펙트에 컬러 슬롯 고정
- 근접전에서 다리 디테일이 묻히지 않도록 하체 명암 대비 강화
- 애니메이션 시 꼬리/귀가 방향 정보를 주는 보조 HUD 역할을 하도록 설계

### 9-3. 표현 톤 가이드(서비스 안전선)
- 매력 포인트는 포즈/실루엣/스타일링으로 전달
- 노출보다는 기능성 장비 + 핏 라인 중심으로 "세련된 매혹" 구현
- 글로벌 연령등급/플랫폼 정책을 고려한 안전한 코스메틱 변주를 기본값으로 유지
